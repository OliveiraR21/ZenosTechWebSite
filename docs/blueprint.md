# **App Name**: Zenos Web

## Core Features:

- Hero Section: Display a visually striking hero section with the Zenos logo, a concise title, and a compelling subtitle.
- Manifesto Section: Showcase the core values of Zenos with a centralized, impactful statement.
- Differentiators: Highlight the three key pillars of Zenos (Agilidade Simples, Inovação Duradoura, Visão Limpa) with minimalist cards.
- The NIKO Factor: Introduce NIKO as a strategic mentor, simplifying operations and guiding scaling through an AI 'tool'.
- Footer: Display the Zenos logo, slogan, and links to LinkedIn, Instagram, and a contact form.

## Style Guidelines:

- Background: Absolute Black (#000000) for primary background with Graphite Gray (#1F1F1F) for sections to add depth.
- Primary: Neon Green (#CCFF00) extracted from the logo, used for CTAs, highlights, and interactive elements.
- Headings: 'Teko' (sans-serif) in uppercase for a technical and solid appearance. Note: currently only Google Fonts are supported.
- Body: 'Inter' (sans-serif) for maximum legibility. Note: currently only Google Fonts are supported.
- Minimalist cards in Graphite Gray (#1F1F1F) to present the three pillars (Agilidade Simples, Inovação Duradoura, Visão Limpa).
- Subtle neon green light animation running across borders of key sections to add a high-tech feel.
